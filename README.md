# ☁️ Implementação de Serviços AWS — Redução de Custos

Relatório técnico de implementação de serviços AWS na empresa fictícia **Abstergo Industries**, desenvolvido como parte do bootcamp de **Engenharia de Dados** na [DIO](https://www.dio.me/).


## 📋 Sobre o Projeto

O objetivo deste projeto foi identificar e documentar **3 serviços AWS** estratégicos para reduzir custos operacionais imediatos, substituindo uma infraestrutura on-premise por soluções em nuvem escaláveis e com modelo de pagamento sob demanda.


## ☁️ Serviços AWS Implementados

| # | Serviço | Foco | Economia Estimada |
|---|---|---|---|
| 1 | **Amazon EC2 + Auto Scaling** | Infraestrutura de servidores sob demanda | ~40% |
| 2 | **Amazon S3 + Intelligent-Tiering** | Armazenamento otimizado por camadas | ~70% |
| 3 | **AWS Lambda** | Processamento serverless sem servidores ociosos | ~60% |


## 💡 Conceitos Abordados

- Migração de infraestrutura **On-Premise → Cloud**
- Modelo de cobrança **Pay-as-you-go**
- **Escalabilidade horizontal** com EC2 Auto Scaling
- **Armazenamento em camadas** com S3 Intelligent-Tiering e S3 Glacier
- **Serverless computing** com AWS Lambda
- Boas práticas de **redução de custos na AWS**
